As a challenge, I designed this site using vanilla HTML and CSS, but a bit of JavaScript did end up being necessary for the night mode feature. I wrote it from scratch in Sublime Text, though these days I'm rolling with VSC.

This site is hosted on Nearly Free Speech, which is a terrific resource for small static site developers. Shoutout to St. Helena, Ascension and Trstan da Cunha, British Overseas Territories, for the .sh domain. I'm certain I'm not the only Josh in the world who's used it.

I'm currently working on ensuring the site passes the accesability standards put forth by the WCAG 2.1, so check back later for that I guess.

Oh, and the font you're looking at is a zesty, readable monospace called Inconsolata.
